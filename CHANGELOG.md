# Changelog

## [0.1.0] — 2026-03-16

### Added
- NRP Identity: universal addressing `nrp://scope/kind/name`
- NRP Manifest: self-describing nodes (observe, act, shield)
- NRP Events: real-time push with severity levels, emergency bypass
- NRP Driver: universal interface (4 methods, ~100 lines per driver)
- Python SDK with full type hints
- Specification documents (Identity, Manifest, Events)
- Examples: hello_ssh.py, multi_node.py
- MIT license
